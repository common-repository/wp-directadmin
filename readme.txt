== Description ==
Connect to Directadmin Control Pannel and show Disk Usage and Bandwidth Usage in WordPress Dashbord using directadmin api.

= Support =
* [طراحی سایت](https://sorendata.com/)
* [تصفیه آب](http://zistpakarad.ir/)


== Installation ==
1. Upload `wp-directadmin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
